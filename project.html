Proposed Team Project: SMS Spam Classifier

Project summary

This project will build a machine learning-based SMS spam classifier (real-time classification). We will experiment with TF-IDF and word embeddings, and try classifiers such as Naive Bayes, Logistic Regression and SVM. The system will include a REST API and an optional web dashboard.

Project details

  • Data preprocessing and feature extraction (TF-IDF / embeddings)
  • Model training and evaluation (accuracy, precision, recall, F1)
  • REST API for real-time classification
  • Web dashboard to display results and performance metrics
